# CRM de Doações 

### 1. Introdução

O CRM de Doações é um sistema web desenvolvido para gerenciar doadores, doações e campanhas de arrecadação.
O sistema permite centralizar informações de apoiadores e acompanhar o histórico de doações, facilitando a gestão das organizações sociais.
Visa, ainda, possibilitar uma melhor gestão e visualização de doações, doadores e prospectos da instituição.

O projeto está sendo executado em parceria com a instituição social, João XXIII.

### 2. Objetivo

O objetivo do sistema é:
- Organizar os dados dos doadores
- Registrar doações
- Controlar campanhas
- Gerar relatórios
- Facilitar a gestão de prospectos

### 3. Tecnologias

- Backend: Java
- Frontend: HTML/CSS/JS
- Banco: MySQL
